# **App Name**: Academic Mirror

## Core Features:

- Content Replication: Replicate the site's layout, spacing, typography, and responsive breakpoints using HTML and CSS to visually match the target site.
- Image Placeholder: Replace images with licensed stock or placeholders with the same composition and aspect ratio as the original site's images.
- Text Replacement: Insert realistic placeholder text (lorem ipsum + short descriptors) in every content area to ensure visual identity while maintaining original text.
- SEO Optimization: Implement SEO meta tags to improve the site's visibility and search engine ranking.
- Accessibility Compliance: Ensure the site is accessible by implementing semantic HTML and ARIA attributes for navigation.
- Firebase Integration: Integrate firebase.json for hosting, with rewrites to index.html for SPA behavior.
- Deployment Instructions: Provide clear instructions for deploying the website using Firebase (firebase init, firebase deploy).

## Style Guidelines:

- Primary color: Dark teal (#008080) to reflect the academic institution's branding.
- Background color: Very light gray (#F5F5F5), nearly white, to provide a clean and modern feel.
- Accent color: A muted orange (#D38635) to highlight key elements.
- Body and headline font: 'PT Sans', a humanist sans-serif combining modernity with warmth, suitable for both headlines and body text.
- Recreate a grid-based layout and responsive breakpoints that mirror the target website's structure, adjusting for desktop, tablet, and mobile views.
- Employ simple, professional icons to represent different sections or features of the website, aligning with the overall design theme.
- Use subtle, tasteful animations for transitions and interactive elements to enhance user engagement without being distracting.
